Release Notes

We're constantly working to improve weather experience from our users. Below, you will find lists of fixes and new features upon each update.


What's New? (7/13/2022)

New and cool stuff

  • Added a genuine homepage. No more immediately opening up to Houston's weather.
  • Added Cypress and Jersey Village weather. Now, you'll know the weather in those locations too.
  • Added some weather icons to hourly weather to make it look more interesting.
  • Removed the decimal point for the temperatures. Instead of 93.45°F, the temperature will now look like 93°F.
  • Changed up the interface design for each location.
  • Changed some other cosmetics and functionality to make the website look and feel generally better.

Operations in the back end

  • Took out some unnecessary code and replaced it with fresh new code. This may or may not help with performance when loading the website.
  • Took out the space background the site originally had, as it made the site look a bit tacky.

What's New? (6/30/2022)

New and cool stuff

  • Added a page specifically for release notes to list everything new and all squashed bugs!
  • Added the ability to add Geoweather to your home screen (iOS only).
    The instructions can be found here.

Hot fix (6/27/2022)

Operations in the back end

Hourly Weather

  • In Hourly Weather, the "Rain chance %" column would consist of massive amounts of significant figures in certain rows (e.g. 14% would show as "14.0000000000...02%"). This has been fixed and no longer occurs.
