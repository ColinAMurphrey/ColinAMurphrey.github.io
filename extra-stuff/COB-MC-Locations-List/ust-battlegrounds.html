Instructions to get started:

Firstly, download and install Forge:

Download Forge

After downloading, run the file and click "OK". The installation will begin.

Next, download the mandatory mods:

Download mods

Unzip the file and place the two mods in the mods folder in your .minecraft folder.

To get to the .minecraft folder, press Win + R and type %appdata%. Then, navigate to the .minecraft folder.

After you have dropped the mods in the mods folder, you should be able to play Minecraft with the mods installed. The IP:port for the server will be in the Discord server. If you cannot find the IP and port, contact me.
